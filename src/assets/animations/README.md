# Lottie Animation Files

## Success Animation

Place your Lottie success animation JSON file here and name it `success.json`.

### How to get a Lottie JSON file:

1. **From LottieFiles.com**:
   - Visit https://lottiefiles.com
   - Search for "success" or "checkmark" animations
   - Download the JSON file
   - Save it as `success.json` in this directory

2. **From After Effects**:
   - Create animation in After Effects
   - Use Bodymovin plugin to export as JSON
   - Save as `success.json`

3. **Recommended animations**:
   - Success checkmark
   - Confirmation animation
   - Success tick animation

### File Location:
```
src/assets/animations/success.json
```

### File Format:
The file should be a valid Lottie JSON file (usually starts with `{"v":"5.x.x","fr":...}`)

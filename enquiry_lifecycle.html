4. All Endpoints Summary

EndpointMethodsPurpose
POST /api/enquiries POST Create new enquiry
GET /api/enquiries/search GET List enquiries with filters/pagination
GET /api/enquiries/:id GET Get single enquiry with Coral[]/Cad[] arrays
PUT /api/enquiries/:id PUT Update status, substatus, assignedTo, clientId
DELETE /api/enquiries/:id DEL Delete enquiry
POST /api/enquiries/:id/upload/coral POST Upload coral design (FormData)
POST /api/enquiries/:id/upload/cad POST Upload CAD design (FormData)
PUT /api/enquiries/:id/upload/:type?version=X PUT Save pricing / Approve version / Reject version / Mark final
POST /api/enquiries/:id/upload/reference POST Upload reference images
GET /api/enquiries/aggregate GET Status statistics & bucket counts
POST /api/validate-image POST Validate image before upload
